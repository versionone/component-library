Used to provide a phone number. Uses a number keypad on mobile devices and restricts keyboard input to
`Space`, `Backspace`, `Shift`, `Left`, `Right`, `Up`, `Down`, `1`, `2`, `3`, `4`, `5`, `6`, `7`, `8`, `9`, `0`, `+`, `-`
