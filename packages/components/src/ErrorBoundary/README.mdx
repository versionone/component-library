import { ErrorBoundary } from './';

Error boundaries catch unexpected errors and enable presenting the user with a user interface that indicates something went unexpectedly wrong.

## Default

<Playground>
  <ErrorBoundary>
    <Example />
  </ErrorBoundary>
</Playground>

## Log to external service

<Playground>
  <ErrorBoundary onError={alert}>
    <Example />
  </ErrorBoundary>
</Playground>

## Custom fallback user interface

<Playground>
  <ErrorBoundary Fallback={Fallback} onError={alert}>
    <Example />
  </ErrorBoundary>
</Playground>
