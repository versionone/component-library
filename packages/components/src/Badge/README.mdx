# Badge

Badges are used to show notification count and call the user's attention to take action on them.

### Comparisions

| Component                      | Purpose / Use Case         |
| :----------------------------- | :------------------------- |
| [Lozenge](/components/Lozenge) | Highlight an item's status |
| [Badge](/components/Badge)     | Counts and tallies         |
| [Tag](/components/Tag)         | Visually label items       |
