# FormField

Groups a `Label` and a Form Control (`TextField`, `Select`, etc) such that the `Label` will trigger the focus of the control.

Aside: Enables full control of the layout of the `Label` and Form Control while preserving browser's natrual behaviour when nesting `inputs` within a `label`
